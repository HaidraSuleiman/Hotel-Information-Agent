# Hotel-Information-Agent
An AI-Powered agent that answers guest questions.

## Features
- Answering questions about amenities
- Answering questions about restaurant hours
- Answering questions about check out times
- Responding to all guests inquiries about the hotel

## Architecture and Technologies
- Azure AI Foundry
- AI Model: gpt-4.1-mini
- Instructions: You are a helpful hotel consierge attending to guest inquiries and concerns. Be nice and have a great attitude to the customer. Keep answres short and precise, if you don't know the answer say so.
